# add
Adds multiple (more than one) matrices or/and numbers

Wiki: [Matrix addition](https://en.wikipedia.org/wiki/Matrix_addition)

```javascript
const xMatrix = [
    [1, 2, 3],
    [1, 2, 3]
]
const yMatrix = [
    [1, 1, 1],
    [1, 1, 1]
]
const result = add(xMatrix, yMatrix)
```
the `result` is
```
[
    [2, 3, 4],
    [2, 3, 4]
]
```
