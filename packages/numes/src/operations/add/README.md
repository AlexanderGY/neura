# add
Adds multiple (more than one) matrices or/and numbers

[Matrix addition](https://en.wikipedia.org/wiki/Matrix_addition)

```javascript
const ndarr1 = [
    [1, 2, 3],
    [1, 2, 3]
]
const ndarr2 = [
    [1, 1, 1],
    [1, 1, 1]
]
const result = add(ndarr1, ndarr2)
```
the `result` is
```
[
    [2, 3, 4],
    [2, 3, 4]
]
```
