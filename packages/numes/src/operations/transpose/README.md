# transpose
Flips a matrix over its diagonal, switching columns to rows

Wiki: [Transpose of a matrix](https://en.wikipedia.org/wiki/Transpose)

```javascript
const ndarr = [
    [1, 2, 3],
    [4, 5, 6]
]

const result = transpose(ndarr)
```

the `result` is

```
[
    [1, 4],
    [2, 5],
    [3, 6]
]
```
